# SfChunkInfo - 区块信息

一个简易的区块扫描插件，可以获取区块中所有 Slimefun 方块的数量

支持 Slimefun 附属以及保护插件(Towny, GriefPrevention 等)

## 使用

```/sfchunkinfo``` - 扫描你所在区块 (权限: **sfchunkinfo.scan**)

```/sfchunkinfo <目标玩家>``` - 扫描该玩家所在区块，无视保护插件 (权限: **sfchunkinfo.scan.others**)
